# exemplo_H2
Projeto desenvolvido na aula de DEV-Ti

## Como rodar: 
Tenha o NodeJS instalado e rode os seguintes comandos 
```
npm install
npm run dev
```

Após isso, basta abrir o link disponibilizado no terminal e o projeto estará rodando! 

## Tecnologias
- HTML / CSS / JavaScript
- Node.Js
- Express
